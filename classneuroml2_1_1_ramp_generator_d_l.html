Detailed Description

Class corresponding to the RampGeneratorDL schema type.

Dimensionless equivalent of rampGenerator . Generates a ramping current after a time delay, for a fixed duration. During this time the dimensionless current steadily changes from startAmplitude to finishAmplitude. Scaled by weight, if set
:param delay: Delay before change in current. Current is baselineAmplitude prior to this. :type delay: time :param duration: Duration for holding current at amplitude. Current is baselineAmplitude after delay + duration. :type duration: time :param startAmplitude: Amplitude of linearly varying current at time delay :type startAmplitude: none :param finishAmplitude: Amplitude of linearly varying current at time delay + duration :type finishAmplitude: none :param baselineAmplitude: Amplitude of current before time delay, and after time delay + duration :type baselineAmplitude: none
