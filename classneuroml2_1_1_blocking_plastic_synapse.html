Detailed Description

Class corresponding to the BlockingPlasticSynapse schema type.

Biexponential synapse that allows for optional block and plasticity mechanisms, which can be expressed as child elements.
:param tauRise: :type tauRise: time :param tauDecay: :type tauDecay: time :param gbase: Baseline conductance, generally the maximum conductance following a single spike :type gbase: conductance :param erev: Reversal potential of the synapse :type erev: voltage
