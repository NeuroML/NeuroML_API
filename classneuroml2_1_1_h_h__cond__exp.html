Detailed Description

Class corresponding to the HH_cond_exp schema type.

Single-compartment Hodgkin-Huxley-type neuron with transient sodium and delayed-rectifier potassium currents using the ion channel models from Traub.
:param gbar_K: :type gbar_K: none :param gbar_Na: :type gbar_Na: none :param g_leak: :type g_leak: none :param e_rev_K: :type e_rev_K: none :param e_rev_Na: :type e_rev_Na: none :param e_rev_leak: :type e_rev_leak: none :param v_offset: :type v_offset: none :param e_rev_E: :type e_rev_E: none :param e_rev_I: :type e_rev_I: none :param cm: :type cm: none :param i_offset: :type i_offset: none :param tau_syn_E: This parameter is never used in the NeuroML2 description of this cell! Any synapse producing a current can be placed on this cell :type tau_syn_E: none :param tau_syn_I: This parameter is never used in the NeuroML2 description of this cell!
