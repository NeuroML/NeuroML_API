Detailed Description

Class corresponding to the ExpOneSynapse schema type.

Ohmic synapse model whose conductance rises instantaneously by ( gbase * weight ) on receiving an event, and which decays exponentially to zero with time course tauDecay
:param tauDecay: Time course of decay :type tauDecay: time :param gbase: Baseline conductance, generally the maximum conductance following a single spike :type gbase: conductance :param erev: Reversal potential of the synapse :type erev: voltage
